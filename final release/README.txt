MARATHON REVIT ADDINS

INSTALLATION INSTRUCTIONS:

1.download the correctly versioned .exe file from this folder.
2.Exit all Autodesk products before running the installer. If you do not, the installer will prompt you to close them anyways.
3.Run the installer
4.Restart Revit
5.A prompt will come up when Revit is launched. Click "Always Load" at this prompt. If it ever shows up again, just keep clicking "Always Load", it will eventually stick. Revit is fickle like that sometimes.

6.The Installation is finished. The addins will appear under the tab called "Marathon Addins" in Revit.
